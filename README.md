# Trader Mini Apps

Este repositório é uma coleção de mini-aplicativos e ferramentas desenvolvidos para dar suporte à área financeira, direta ou indiretamente. O objetivo é fornecer utilitários eficientes para análise, automação e gestão em mercados de ações, derivativos e criptomoedas.

## Projetos e Módulos

### 🤖 [AI Automation](./ai_automation/README.md)
Um motor de agência de conteúdo e SEO orientado por IA. Este projeto automatiza o pipeline desde a pesquisa de mercado até a geração de conteúdo otimizado, ajudando na presença digital e marketing de serviços financeiros.

> **Nota:** Para detalhes sobre instalação, configuração e funcionalidades do módulo de automação, consulte o [README específico do ai_automation](./ai_automation/README.md).
